## 使用方法

1. Fork 或 Use this Template 本仓库。
2. 创建好仓库后，启用 GitHub Actions。
3. 按需更改 config.txt 中的内容（非必要步骤）
4. 将你要下载的 YouTube 视频的地址放入 playlist.txt 中，每行限一个视频链接。
5. Actions 运行好后会将下载好的视频打包成 downloads.tar.gz 发布到  Release 中，下载即可。
